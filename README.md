# 🎨 Turtle Spirograph Generator

A mesmerizing, terminal-free spirograph generator built with Python's built-in `turtle` module. This application features 15 unique, algorithmically generated mathematical patterns, complete with dynamic HSV color gradients and a custom interactive UI.

![Spirograph Demo](assets/demo.gif) *(Note: Add your GIF path here)*

## ✨ Features
* **15 Unique Mathematical Patterns:** Includes Butterfly, Galaxy Spiral, Nautilus Shell, Fractal Snowflakes, and more.
* **Dynamic Color Mapping:** Utilizes `colorsys` to generate smooth, psychedelic HSV gradients based on algorithmic math.
* **Interactive GUI:** Custom built-in menu system allowing users to select patterns without a terminal.
* **State Saving:** Automatically remembers and displays your last drawn pattern using JSON handling.
